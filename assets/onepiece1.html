Fix Embed Blocked Error

If you are seeing an error on the player "Embed Blocked", you can get past this in 3 ways :

1. Switch Browser (Recommended - Firefox)

2. Use a Different Server to Stream

3. (If you still prefer to use Chrome or any browser that uses chromium source code) Copy and Paste this into your Browser URL/Address Bar -

chrome://settings/content/pdfDocuments

And then Enable "Download PDF files instead of automatically opening them in Chrome". Reload the page, and you won't be seeing the Embed Blocked Error again. (Don't worry you don't need to do this each time you open your browser)
